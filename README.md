
# Titanic Data Cleaning & Analysis
This project demonstrates my ability to clean and preprocess raw data using Python's **Pandas** library.

### Key Data Cleaning Steps:
* **Missing Values:** Identified gaps using `df.isnull().sum()`.
* **Age Column:** Filled 177 missing values with the **mean** age to preserve data.
* **Cabin Column:** Dropped the column entirely due to 687 missing values.
* **Embarked Column:** Removed 2 rows with missing values to ensure data integrity.
